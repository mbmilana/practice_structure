## Описание программы
Программа сортирует записи о самолетах с использованием индексной сортировки линейным методом по высотным эшелонам

Данные о самолетах хранятся в файле, где в первой строке записано число:
- **n** - количество записей о летательных аппаратах

В следующих n строках перечислены записи, каждая из которых имеет структуру:\
`Марка ЛА    Бортовой номер    Высота ЛА`


Выводится таблица записей, отсортированных по высотным эшелонам, каждая запись имеет структуру:\
`№   Марка ЛА    Бортовой номер    Высота    Эшелон    ID`

где:
- **ID** - номер ЛА в исходной таблице записей
- **№** - порядковый номер ЛА в отсортированной таблице записей

## Компиляция и запуск
Для корректного вывода записей о ЛА на русском языке необходимо работать с кодировкой Windows 1251.\
Для VSCode необходимо сделать:\
- сохранить файл **main.cpp** и **файл входных данных** в кодировке Windows 1251
- настроить кодировку терминала (powershell), если не настроена Windows 1251: `[Console]::OutputEncoding = [System.Text.Encoding]::GetEncoding("windows-1251")`

### Компиляция
`g++ main.cpp -o main`
## #Запуск
`./main`
